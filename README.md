
### Usage
./fractol [mandelbrot | julia | burningship]  
keyboard ```c``` for next colorscheme  
keyboard ```s``` and ```d``` for manual iteration depth  
```left click``` drag or ```arrows``` to move  
```scroll``` to zoom  
zoom works until double rounding error
